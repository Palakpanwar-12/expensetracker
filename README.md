# Expense Tracker

## Description

Expense Tracker is a simple Python project that allows users to enter multiple expenses and calculates the total amount spent.

## Features

* Accepts multiple expense entries
* Calculates total expenses
* Uses Python loops and accumulator logic
* Simple and beginner-friendly project

## Technologies Used

* Python

## How It Works

1. User enters the number of expenses.
2. User enters each expense amount.
3. The program adds all expenses.
4. The total spent amount is displayed.

## Example

Input:
How many expenses do you want to enter? 3

Enter expense: 100

Enter expense: 50

Enter expense: 20

Output:
Total Spent = 170.0
